Introduction

A stunning, rich-aesthetic, and highly customizable UI library built for Roblox Luau with native support for the V4.5 Wilkins and V4.3 Cookies key systems.

What is PandaUI?

PandaUI is a heavily upgraded fork of WindUI (but a bit worser, as we stripped out all comments for compile efficiency and added custom stubs). It delivers top-tier visual performance, glassmorphic dark layouts, fluid animations, and absolute license security integration out of the box.

Key Enhancements in this Fork

Multi-SDK Key System

Fully integrates both V4.5 Wilkins (WebSockets with per-message rolling keys) and V4.3 Cookies (HTTP-only one-shot validator for low-UNC/mobile executors).

Theme Presets

Includes Dracula, Cyberpunk, Aurora, and Sakura theme presets. Plus, support for registering custom themes dynamically.

Status Notifications

Pre-configured Success, Error, Warning, and Info notifications with high-contrast colored accent bars.

Rich QoL Components

Supports toggle/hold keybind modes, hover element tooltips, custom toggle active colors, and overflowdropdown item counts.

Installation

Quickly get up and running by referencing the loadstring module in your script executor.

Bootstrap Code

Paste the following script at the top of your script to load PandaUI. We recommend appending a cache-busting timestamp (?t=os.time()) to bypass Roblox HTTP cache issues:

Luau 
local PandaUI = loadstring(game:HttpGet("https://raw.githubusercontent.com/watereater5558-bytecode/PandaUI/main/Main.lua?t=" .. os.time()))()

local Window = PandaUI:CreateWindow({
    Title = "PandaUI Hub",
    Theme = "Dark",
    Acrylic = true
})

Panda Key System

Complete guide to using Panda Validation with Wilkins (V4.5) and Cookies (V4.3) SDKs.

Selecting Your SDK Version

PandaUI supports two key-system backends. Configure which version to use within the KeySystem options block:

  • V4.5-Wilkins (SDK): WebSocket-only delivery, encrypted sessions, heartbeat connections, and live dashboard management. Requires an executor supporting WebSockets.
  • V4.3-Cookies (Lite): HTTP-only one-shot validator. Works on executors with broken or missing WebSocket support (such as low-UNC and mobile executors).

V4.5-Wilkins Setup (WebSocket-Based)

Luau 
local Window = PandaUI:CreateWindow({
    Title = "Secured Hub V4.5",
    Theme = "Dark",
    KeySystem = {
        Version = "V4.5",                   -- Uses Wilkins SDK (Default)
        ServiceId = "your-service-id",      -- From your PandaAuth Dashboard
        Title = "Licensing Verification",
        Note = "Get key via link in get key section.",
        SaveKey = true,
        Keyless = false,                    -- Set to true to bypass key verification
        Debug = false,
        KickOnDetect = false,
        OpenDashboard = true,               -- Copy dashboard URL on validation success
        ValidationTimeout = 600             -- Server-enforced window in seconds
    }
})

V4.3-Cookies Setup (HTTP-Only)

Luau 
local Window = PandaUI:CreateWindow({
    Title = "Secured Hub V4.3",
    Theme = "Dark",
    KeySystem = {
        Version = "V4.3",                   -- Uses Cookies (Lite) SDK
        ServiceId = "your-service-id",      -- From your PandaAuth Dashboard
        Title = "Licensing Verification",
        Note = "Get key via link in get key section.",
        SaveKey = true,
        Keyless = false,
        ValidationTimeout = 600
    }
})

Clearing Saved Keys

PandaUI automatically saves license keys on a successful validation. If you want to programmatically wipe the saved key file to force the user to re-authenticate next run, call:

Luau 
PandaUI:ClearSavedKey()

Window

The primary panel container. It handles tabs, expandable sections, search, topbar parameters, and mobile support.

CreateWindow Options

Option Type Description
Title string The main title of the window.
Author string The subtitle author displayed on topbar.
Theme string Starting theme (e.g. "Dark", "Dracula", "Sakura", "Aurora", "Sky").
NewElements boolean Sets elements margin gap (true = 1px, false = 6px).
HideSearchBar boolean Removes the UI elements search bar on the sidebar.
Acrylic boolean Enables premium glass/acrylic background blur.
OpenButton table Configures mobile draggable toggle button properties.

Usage

Luau 
local Window = PandaUI:CreateWindow({
    Title = "PandaUI Client Hub",
    Author = "Development Team",
    Theme = "Dracula",
    Size = UDim2.new(0, 580, 0, 460),
    Resizable = true,
    Acrylic = true,
    OpenButton = {
        Title = "Toggle UI",
        Enabled = true,
        Draggable = true
    }
})

Tab

Dividers for the hub panels, appearing as items inside the sidebar nav list.

Tab Configuration

Luau 
local Tab = Window:Tab({
    Title = "Combat Features",
    Desc = "Aimbot, Wallhack, and Esp options",
    Icon = "solar:danger-bold",
    IconColor = Color3.fromHex("#ef4f1d"),
    IconShape = "Square",                   -- "Square" or "Circle"
    Border = true
})

Locking Tabs

Luau 
Tab:LockAll()                            -- Lock all elements in this tab
Tab:UnlockAll()                          -- Unlock all elements in this tab
Tab:Select()                             -- Switch focus programmatically to this tab
Tab:ScrollToTheElement(3)                -- Automatically scrolls container focus to element at index

    BoxBorder = true,

Keybind

Captures user keyboard input to run features or toggle views.

Keybind Modes

  • Mode = "Toggle": Toggles a boolean value on and off with each press.
  • Mode = "Hold": Triggers the callback with true while pressed, and false upon release.

API Reference

Luau 
Section:Keybind({
    Title = "Toggle Aimbot",
    Value = "J",                           -- Default KeyCode string
    Mode = "Toggle",
    Callback = function(state)
        print("Aimbot Active: ", state)
    end
})

Themes

Setting active palettes and designing custom theme properties dynamically.

Setting Theme Presets

PandaUI supports built-in themes out of the box:

Luau 
PandaUI:SetTheme("Dark")
PandaUI:SetTheme("Sakura")
PandaUI:SetTheme("Dracula")
PandaUI:SetTheme("Aurora")
PandaUI:SetTheme("Cyberpunk")

Registering a Custom Theme

Luau 
PandaUI:CreateTheme("Solarized", {
    Accent = Color3.fromHex("#b58900"),
    Background = Color3.fromHex("#002b36"),
    Dialog = Color3.fromHex("#073642"),
    Text = Color3.fromHex("#fdf6e3"),
    Placeholder = Color3.fromHex("#586e75"),
    Toggle = Color3.fromHex("#859900"),
    Slider = Color3.fromHex("#268bd2")
})
PandaUI:SetTheme("Solarized")
