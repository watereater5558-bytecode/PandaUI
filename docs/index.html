Introduction

A stunning, rich-aesthetic, and highly customizable UI library built for Roblox Luau with native support for the V4.5 Wilkins key system.

Why choose PandaUI?

PandaUI provides top-tier visual performance, clean glassmorphic components, fluid animations, and absolute security integration with the self-hosted WebSocket-based V4.5 Wilkins SDK right out of the box.

Key Features

Native Wilkins V4.5

Secure WebSocket verification with per-message rolling keys, automatic saved-key flow, and kick-on-tamper protection.

Curated Themes

Dozens of premium pre-built palettes (Midnight, Mellowsi, Sky, Monokai, Sakura, etc.) and complete customization control.

Hardware Lock (HWID)

Built-in integration to handle and lock license keys directly to hardware profiles.

Zero Comments

Super clean codebase optimized for fast compilation and memory-efficient execution in exploit runtimes.

Installation

Quickly get up and running by referencing the loadstring module in your script executor.

Bootstrap Code

Paste the following script at the top of your exploit injector to load PandaUI:

Luau 
local PandaUI = loadstring(game:HttpGet("https://raw.githubusercontent.com/your-repo/PandaUI/main/src/Init.lua"))()
local Window = PandaUI:CreateWindow({
    Title = "PandaUI Hub",
    Theme = "Dark",
    Acrylic = true
})

Wilkins Key System Integration

PandaUI embeds the self-hosted V4.5 Wilkins Lua SDK so you don't have to build any validation UI or connections from scratch.

Usage Example

Configure the key system properties within your window configuration. You can easily make it Keyless or clear the saved keys programmatically.

Luau 
local PandaUI = loadstring(game:HttpGet("https://raw.githubusercontent.com/your-repo/PandaUI/main/src/Init.lua"))()

local Window = PandaUI:CreateWindow({
    Title = "Secured Hub",
    Theme = "Dark",
    KeySystem = {
        ServiceId = "your-service-id-here",
        Title = "Licensing System",
        Note = "Please obtain a key from our dashboard link below.",
        SaveKey = true,
        Keyless = false,
        Debug = false,
        KickOnDetect = false
    }
})

Programmatic Deletion

If you want to clear the saved license key so the user is prompted fresh on next execution, call the clear method:

Luau 
PandaUI:ClearSavedKey()

Window

The core panel container that holds tabs, buttons, section headers, search, and navigation.

API Reference

Luau 
local Window = PandaUI:CreateWindow({
    Title = "Dashboard",
    Author = "Panda Devs",
    Theme = "Sky",
    Size = UDim2.new(0, 580, 0, 460),
    Resizable = true,
    Acrylic = true
})

Section

Groups elements together inside a tab with a clean visual header.

API Reference

Luau 
local Tab = Window:AddTab({ Title = "Combat", Icon = "sword" })
local Section = Tab:AddSection({ Title = "Aimbot Settings" })

Tab

Creates a separate view/category within the sidebar interface.

API Reference

Luau 
local Tab = Window:AddTab({
    Title = "Settings",
    Icon = "settings"
})

Button

A basic clickable control to fire specific functions.

API Reference

Luau 
Section:AddButton({
    Title = "Reset Character",

Checkbox

A toggleable option box designed for simple state configurations.

API Reference

Luau 
Section:AddCheckbox({
    Title = "Auto-Loot Items",
    Default = false,
    Callback = function(value)

Code View

Formats code scripts or output text with custom mono font wrapping inside the interface.

API Reference

Luau 
Section:AddCode({

Dropdown

Displays a list of options that a user can select from.

API Reference

Luau

    Multi = false,
    Options = { "Closest", "Lowest HP", "Random" },
    Default = "Closest",
    Callback = function(selected)
        print("Selected mode: ", selected)
    end
})

Popup / Dialog

Centered, premium floating modual requiring explicit user focus and button callback confirmation.

API Reference

Luau 
PandaUI:Popup({
    Title = "Confirm Action",
    Content = "Are you sure you want to load custom scripts?",
    Buttons = {
        {
            Title = "Load",
            Icon = "check",
            Variant = "Primary",
            Callback = function() print("Loading") end
        },
        {
            Title = "Cancel",
            Icon = "x",
            Variant = "Secondary",
            Callback = function() print("Cancelled") end
        }
    }
})

Themes

Configure beautiful custom gradients, element borders, and colorways dynamically.

Setting Theme

Luau 
PandaUI:SetTheme("Midnight")

Registering a Custom Theme

Luau 
PandaUI:CreateTheme("Minty", {
    Accent = Color3.fromHex("#00f5a0"),
    Background = Color3.fromHex("#051912"),
    Dialog = Color3.fromHex("#082b1f"),
    Text = Color3.fromHex("#e6fff5"),
    Placeholder = Color3.fromHex("#3a8e72")
})
PandaUI:SetTheme("Minty")
